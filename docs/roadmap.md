## 0. 제품 포지셔닝 (한 줄 요약)

"보험주 투자자용 FnGuide"는 미끼, 실체는 **B2B 규제데이터 피드 (DaaS)**.
- Retail TAM 천장 낮음 (상장 보험사 9개, 저회전 배당주). B2C 마케팅에 CAC 태우면 실패.
- 진짜 수익원: 보험·재보험사 실무자 (객단가 수십~수백 배). FnGuide 단말 3,500만원의 1/5~1/10.
- 텀윈드: 2025下 기본자본 K-ICS 규제 + 자본성증권 발행 폭증 (23년 3.15조 → 24년 8.66조) + 재보험 솔벤시 거래 확대.

### 1A-2. 재보험 영업 관점 우선 추가지표 (2026-05-28 사용자 우선순위)

현재 사이트 최대 gap. 요구자본 합산액(treemap 크기)만으론 부족 — "안을 뜯어야" 재보험 솔루션이 결정됨.

| 우선 | 지표 | 재보험 시사점 / 비고 | 출처 |
|------|------|------|------|
| 🔴 | **K-ICS 요구자본 위험액 세부분해** | 해지위험액(대량해지)→Mass-lapse / 사망·장해·질병→YRT / 시장위험(금리)→공동재보험 / 일반보험 대재해→Cat XL. **핵심 가치제안 완성점** | K-ICS 경영공시 |
| 🔴 | **RA(위험조정) 규모·신뢰수준** | CSM과 세트. RA 신뢰수준 낮은 회사(예 75%)는 보수성 낮아 향후 CSM 잠식 리스크. IFRS17 워터폴 옆 패널 | DART 주석 |
| 🔴 | **P&L 보험손익 vs 투자손익 분해** | 기존 "보험손익(IM) 분해"는 보험마진 내부분해 — 별개. 순이익이 보험서비스 vs 투자로 갈리는 비중 = 수익 질. 투자손익 의존 高 = 금리하락기 경고 | DART/IR |
| 🔴 | **출재보험료 / 종목별 출재율** | 재보험사 직접 영업 활용도 高. 출재율 낮은 종목 = 재보험 확대 여력 시그널 | 경영공시(분기) |
| 🔴 | **유지율(13·25회차)** | 경영공시 표준 항목(수집 쉬움). 유지율 하락 + 대량해지위험액 大 = mass-lapse 수요 이중확인 | 경영공시 |
| 🔴 | **운용자산이익률** | 회사 간 격차(예 메리츠 3.58% vs 현대 2.60%) 투자자 비교지표 | IR(분기) |

**🟡 후순위 / 제거 결정 (2026-05-28):**
- ~~BS 스냅샷 표~~ → **제거**. DART 직접 열람 수준, 시안성 대비 효익 低.
- ~~파생 KPI 카드 (런웨이·의존도·소진속도·NB교체)~~ → **아카이빙** (`docs/archived_metrics.md`). 원천 지표(CSM 잔액·상각액·NB CSM) 직접 노출이 B2B 신뢰 ↑ → **원천지표 카드 신설**이 후속 과제.
- ΔCSM 민감도 → 유지하되 후순위. "금리충격 시 K-ICS 비율 변동"(금리위험액에서 유도)이 재보험 영업엔 더 직접적.

**확정 (2026-05-28 추가검토):**
- **요구자본 위험액**: 보험위험(사망~사업비·대재해) 기게시. **금리위험액**만 추가 display 후보(공동재보험·듀레이션갭 신호). 그 외 시장위험 하위는 display X — 단 §1B 분산효과 validation용으로 **전체 파싱**.
- **RA 규모**: SKIP 확정. K-IFRS17 신뢰수준 75% 고정 → 금액은 CSM/BEL/보험위험액에서 결정론적 추론, 재보험 트리거 아님, 방금 아카이빙한 파생 proxy와 동류. (RA 해제 라인을 CSM 워터폴에 원천숫자로 끼우는 정도만 선택지.)
- **채택 (재보험 영업 직결)**: GA 채널비중(역선택 선행지표), 위험손해율(⚠️ 공시-실무 왜곡 명시 필요), **재보험 현황**(출재보험료 비중·출재 CSM 규모·원수vs출재 마진갭), **해지율(13/25/37회차)** — 단독은 약하나 mass-lapse 재보험 수요·역선택 스코어의 핵심 입력으로 채택.

### 1B. 데이터 정합성 보정 로직 (🔴 B2B 필수)

- 연결/별도 reconciliation, 잠정/확정 불일치 주석 (Deloitte 지적: 2024 공식공시 ≠ 2023 예비공시)
- 시점 불일치 주석 (예: 신계약CSM FY24 numerator vs 월납환산 1Q25 denominator — 이미 발견된 한화생명 케이스)
- B2B는 숫자 1개 틀리면 신뢰 즉사 → 정합성 로직이 제품 신뢰의 핵심
- **시장위험액 분산효과 validation (2026-05-28)**: 보험위험(사망~사업비)을 분산효과 sqrt로 교차검증했듯, 시장위험 하위위험액 전체(금리·주식·부동산·외환·자산집중 등)를 파싱해 합계·분산효과 정합성 룰 신설 (화면 노출 X, 데이터 신뢰용). 금리위험액만 별도 display 후보(§1A-2).

### 1C. 장래 프로젝션 엔진 (해자)

- **CSM 상각 스케줄 → 장래 CSM 잔액 projection** (F6 yearly granularity 선행)
- **당기순이익 projection** (GLM 내부모형 논문 = 초석. 리치고식 ML 예측 서비스 사례)
- **K-ICS 비율 변동 추이** (forward sim 이미 v3, 자본성증권 콜 차감 + SCR 경과조치 interp)
- → retail이 못 믿는 IFRS17 순이익을 정량 예측 = 카피 불가 프리미엄 정당화

### 1D. "재보험 외 답 없는 원수사" 스크리너 (재보험 leads — ⚠️ R0 선결 후)

- K-ICS 130% 이하 → 기발행 후순위채 상환 곤란
- 기본자본 인정한도 초과 → 추가 신종/후순위 발행 불가
- 당기순이익 저조 + 이자비용 과다 → 추가 발행 곤란
- 위 3필터 교집합 = 재보험만이 해법인 원수사 발굴

### 1E. 규제 뉴스 피드 (사용자 키워드 피드백 랭킹)

다른 프로젝트에서 검증된 기능 이식. 최근 1주 뉴스 스크래핑 → IFRS17·K-ICS·Solvency 등 **규제 관련 뉴스** 필터링 → 사용자 피드백으로 **의미있는 키워드일수록 상위 노출**(학습형 랭킹).
- 가치: 규제 텀윈드 제품에 자연스러운 engagement·retention 훅. Phase 3 콘텐츠 자동화 루프(데이터 업데이트 감지)의 뉴스 버전.
- 구현 메모: 스크래핑 소스(언론·금융위/금감원 보도자료·생손보협회) + 키워드 가중 랭킹(사용자 클릭/피드백 학습). 자동발행 아님, 큐레이션 피드.

이해상충 **소멸** (코리안리는 EU/일본 원수사와 재보험 거래 안 함) + TAM 10배 + 선점 가능한 공백.

| 항목 | 한국 | EU (Solvency II) |
|------|------|-------------------|
| 대상 보험사 | ~60 | ~3,000+ |
